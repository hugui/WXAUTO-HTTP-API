辅助功能

说明：辅助功能主要包括登录相关功能。标记为Plus版的功能需要wxautox库支持。
POST /api/auxiliary/login/auto Plus版 - 自动登录微信
接口说明

自动登录微信，仅对可自动登录的微信有效。需要wxautox库支持。

请求参数
参数名 类型 必需 说明
wxpath string 否 微信客户端路径，不提供时自动检测
timeout number 否 登录超时时间（秒），默认10秒
请求示例
curl -X POST http://localhost:5000/api/auxiliary/login/auto \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "wxpath": "D:/Program Files/Tencent/WeChat/WeChat.exe",
    "timeout": 10
  }'
响应格式
{
  "code": 0,
  "message": "自动登录执行完成",
  "data": {
    "wxpath": "D:/Program Files/Tencent/WeChat/WeChat.exe",
    "timeout": 10,
    "login_result": true,
    "success": true
  }
}
注意事项
  • 仅对可自动登录的微信有效
  • 如果不提供微信路径，系统会自动检测常见安装位置
  • 需要wxautox库支持
POST /api/auxiliary/login/qrcode Plus版 - 获取登录二维码
接口说明

获取登录二维码，返回base64编码的二维码图片，可直接在网页中显示。需要wxautox库支持。

请求参数
参数名 类型 必需 说明
wxpath string 否 微信客户端路径，不提供时自动检测
请求示例
curl -X POST http://localhost:5000/api/auxiliary/login/qrcode \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"wxpath": "D:/Program Files/Tencent/WeChat/WeChat.exe"}'
响应格式
{
  "code": 0,
  "message": "获取登录二维码成功",
  "data": {
    "wxpath": "D:/Program Files/Tencent/WeChat/WeChat.exe",
    "qrcode_path": "/path/to/qrcode.png",
    "qrcode_base64": "iVBORw0KGgoAAAANSUhEUgAA...",
    "qrcode_data_url": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA...",
    "mime_type": "image/png"
  }
}
响应字段说明
  • qrcode_path - 二维码图片文件路径
  • qrcode_base64 - 二维码图片的base64编码
  • qrcode_data_url - 可直接在HTML中使用的data URL格式
  • mime_type - 图片MIME类型
注意事项：
  • 登录相关功能需要wxautox库支持（Plus版）
  • 自动登录功能仅对可自动登录的微信有效
  • 如果不提供微信路径，系统会自动检测常见的安装位置
  • 二维码图片会以base64格式返回，可直接在网页中显示
