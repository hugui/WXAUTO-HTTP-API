好友管理

GET /api/friend/get-details Plus版 - 获取好友详情
接口说明

获取好友的详细信息列表。

响应格式
{
    "success": true,
    "message": "获取好友详情成功",
    "data": {
        "friends": [
            {
                "name": "好友1",
                "remark": "备注名",
                "avatar": "头像路径"
            }
        ]
    }
}
POST /api/friend/add-new-friend Plus版 - 添加新好友
接口说明

发送好友申请。

请求参数
参数名 类型 必需 说明
username string 是 用户名或微信号
message string 否 申请消息
响应格式
{
    "success": true,
    "message": "好友申请已发送",
    "data": {
        "username": "test_user",
        "status": "sent"
    }
}

                        POST
                        /api/auxiliary/new-friend/accept
                        Plus版
                        - 接受好友申请
                    

                    

                        
                        

                            
接口说明

                            
接受新好友申请，可以设置备注和标签。需要wxautox库支持。


                            
请求参数

                            

                                
                                    

                                        参数名
                                        类型
                                        必需
                                        说明
                                    

                                
                                
                                    

                                        friend_name
                                        string
                                        是
                                        申请人名称
                                    

                                    

                                        remark
                                        string
                                        否
                                        好友备注名称
                                    

                                    

                                        tags
                                        array
                                        否
                                        好友标签列表
                                    

                                
                            


                            
请求示例

                            


curl -X POST http://localhost:5000/api/auxiliary/new-friend/accept \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "friend_name": "张三",
    "remark": "同事",
    "tags": ["工作", "同事"]
  }'

                            


                            
响应格式

                            


{
  "code": 0,
  "message": "接受好友申请成功",
  "data": {
    "friend_name": "张三",
    "remark": "同事",
    "tags": ["工作", "同事"],
    "result": "success"
  }
}

                        /api/auxiliary/new-friend/reject
                        Plus版

接口说明

                            
拒绝新好友申请。需要wxautox库支持。


                            
请求参数

                            

                                
                                    

                                        参数名
                                        类型
                                        必需
                                        说明
                                    

                                
                                
                                    

                                        friend_name
                                        string
                                        是
                                        申请人名称
                                    

                                
                            


                            
请求示例

                            


curl -X POST http://localhost:5000/api/auxiliary/new-friend/reject \
