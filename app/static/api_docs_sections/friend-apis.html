好友管理

GET /api/friend/get-details Plus版 - 获取好友详情
接口说明

获取好友的详细信息列表。

响应格式
{
    "success": true,
    "message": "获取好友详情成功",
    "data": {
        "friends": [
            {
                "name": "好友1",
                "remark": "备注名",
                "avatar": "头像路径"
            }
        ]
    }
}
POST /api/friend/add-new-friend Plus版 - 添加新好友
接口说明

发送好友申请。

请求参数
参数名 类型 必需 说明
username string 是 用户名或微信号
message string 否 申请消息
响应格式
{
    "success": true,
    "message": "好友申请已发送",
    "data": {
        "username": "test_user",
        "status": "sent"
    }
}
