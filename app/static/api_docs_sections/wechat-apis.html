WeChat扩展

GET /api/wechat/get-session 基础版 - 获取会话列表
接口说明

获取当前会话列表。

响应格式
{
    "success": true,
    "message": "获取会话列表成功",
    "data": {
        "sessions": [
            {
                "name": "文件传输助手",
                "last_message": "最后一条消息",
                "time": "12:00"
            }
        ]
    }
}
POST /api/wechat/chat-with 基础版 - 打开聊天窗口
接口说明

打开与指定联系人的聊天窗口。

请求参数
参数名 类型 必需 说明
who string 是 联系人名称
exact boolean 否 是否精确匹配，默认false
响应格式
{
    "code": 0,
    "message": "打开聊天窗口成功",
    "data": {
        "who": "文件传输助手",
        "result": "测试文件传输助手"
    }
}
GET /api/wechat/get-sub-windows Plus版 - 获取子窗口
接口说明

获取微信的子窗口列表。

响应格式
{
    "code": 0,
    "message": "获取所有子窗口成功",
    "data": {
        "sub_windows": [
            {
                "who": "文件传输助手",
                "window_id": "window_001"
            }
        ]
    }
}
