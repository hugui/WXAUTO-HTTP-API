消息操作

POST /api/message/forward 基础版 - 转发消息
接口说明

转发消息给指定联系人或群组。

请求参数
参数名 类型 必需 说明
message_id string 是 要转发的消息ID
to_contact string 是 转发目标联系人
响应格式
{
    "success": true,
    "message": "消息转发成功",
    "data": {
        "message_id": "msg_001",
        "to_contact": "文件传输助手",
        "forward_time": "2024-01-01 12:00:00"
    }
}
POST /api/message/download 基础版 - 下载文件
接口说明

下载消息中的图片或文件到本地。

请求参数
参数名 类型 必需 说明
message_id string 是 包含文件的消息ID
save_path string 否 保存路径，默认为下载文件夹
响应格式
{
    "success": true,
    "message": "文件下载成功",
    "data": {
        "message_id": "msg_001",
        "file_path": "C:/Downloads/image.jpg",
        "file_size": "1.2MB"
    }
}
POST /api/message/click 基础版 - 点击消息
接口说明

点击指定的消息元素。

请求参数
参数名 类型 必需 说明
who string 是 聊天对象名称
message_id string 是 消息ID
响应格式
{
    "success": true,
    "message": "消息点击成功",
    "data": {
        "who": "文件传输助手",
        "message_id": "msg_001"
    }
}

POST /api/auxiliary/session/click 基础版 - 点击会话
接口说明

点击会话列表中的指定会话，用于切换到指定的聊天窗口。

请求参数
参数名 类型 必需 说明
session_name string 是 要点击的会话名称
请求示例
curl -X POST http://localhost:5000/api/auxiliary/session/click \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"session_name": "文件传输助手"}'
响应格式
{
  "code": 0,
  "message": "点击会话成功",
  "data": {
    "session_name": "文件传输助手"
  }
}
