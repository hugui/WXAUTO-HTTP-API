朋友圈

POST /api/moments/open Plus版 - 打开朋友圈
接口说明

打开微信朋友圈界面。

响应格式
{
    "success": true,
    "message": "朋友圈已打开",
    "data": {
        "status": "opened",
        "timestamp": "2024-01-01 12:00:00"
    }
}
GET /api/moments/get-messages Plus版 - 获取朋友圈消息
接口说明

获取朋友圈的消息列表。

请求参数
参数名 类型 必需 说明
limit number 否 获取数量，默认10
响应格式
{
    "success": true,
    "message": "获取朋友圈消息成功",
    "data": {
        "messages": [
            {
                "author": "张三",
                "content": "今天天气真好",
                "time": "2024-01-01 12:00:00"
            }
        ]
    }
}
