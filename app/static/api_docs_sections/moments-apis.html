朋友圈

POST /api/moments/open Plus版 - 打开朋友圈
接口说明

打开微信朋友圈界面。

响应格式
{
    "success": true,
    "message": "朋友圈已打开",
    "data": {
        "status": "opened",
        "timestamp": "2024-01-01 12:00:00"
    }
}
GET /api/moments/get-moments Plus版 - 获取朋友圈内容
接口说明

获取朋友圈的动态内容。

请求参数
参数名 类型 必需 说明
n number 否 获取数量，默认10
timeout number 否 超时时间（秒），默认10
响应格式
{
    "code": 0,
    "message": "获取朋友圈内容成功",
    "data": {
        "moments": [
            {
                "author": "好友名称",
                "content": "朋友圈文字内容",
                "time": "2小时前",
                "images": [],
                "likes": [],
                "comments": []
            }
        ]
    }
}
POST /api/moments/like Plus版 - 点赞朋友圈
接口说明

对指定的朋友圈动态进行点赞操作。

请求参数
参数名 类型 必需 说明
moment_index number 是 朋友圈索引（从0开始）
like boolean 否 是否点赞，true为点赞，false为取消点赞，默认true
响应格式
{
    "code": 0,
    "message": "点赞成功",
    "data": {
        "moment_index": 0,
        "result": "点赞结果"
    }
}
POST /api/moments/comment Plus版 - 评论朋友圈
接口说明

对指定的朋友圈动态发表评论。

请求参数
参数名 类型 必需 说明
moment_index number 是 朋友圈索引（从0开始）
text string 是 评论内容
响应格式
{
    "code": 0,
    "message": "评论成功",
    "data": {
        "moment_index": 0,
        "text": "评论内容",
        "result": "评论结果"
    }
}
