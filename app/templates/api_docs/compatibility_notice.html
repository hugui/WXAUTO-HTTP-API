兼容性接口说明

注意：以下接口为兼容性接口，主要用于向后兼容。建议新项目优先使用上述Chat类功能接口。

兼容性接口包括：

  • 消息功能 (/api/message/) - 提供基础的消息发送和接收功能
  • 聊天窗口 (/api/chat-window/) - 提供聊天窗口相关操作

这些接口功能与Chat类接口重复，但保留用于兼容现有项目。
