Chat类功能 推荐使用

说明：Chat类API是基于wxauto库核心设计的主要接口，提供完整的聊天窗口操作功能，推荐优先使用。
POST /api/chat/show 基础版 - 显示聊天窗口
接口说明

显示指定联系人或群组的聊天窗口，相当于在微信中点击打开聊天。

请求参数
参数名 类型 必填 说明
who string 是 聊天对象名称
响应示例
{
    "code": 0,
    "message": "显示聊天窗口成功",
    "data": {
        "who": "测试群"
    }
}

            POST
            /api/chat/send-message
            基础版
            - 发送消息
        

        

            
            

                
接口说明

                            参数名
                            类型
                            必填

                            who
                            string
                            是
                            聊天对象名称
                        

                        

                            message
                            string
                            是
                            消息内容
                        

                        

                            at_list
                            array
                            否
                            @的用户列表
